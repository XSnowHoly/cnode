### 功能
- [x] 首页列表，上拉加载
- [x] 主题详情，回复，点赞
- [x] 消息列表
- [x] 消息提醒

- [x] 退出
- [x] 关于

## 运行项目

``` bash
# step1
git clone git@github.com:PeterCheng1/cnode.git

# step2
npm install

# step3
npm run dev

# step4如果需要打包部署
npm run build

# step4如果需要打包部署
npm run build --report
```
